COTOHA API 性能テスト





使い方

セレクトボックスから使用したいAPI機能を選択し、上のテキストボックスに解析させたい文章を入力して「送信する」ボタンを押してください。

文章の解析結果が下のテキストボックスに表示されます。

*1 プルダウンを選択するとサンプル文が表示されます。そのまま送信すると適切な解析結果が得られます。

*2 解析には少し時間がかかります。また、1000kb以上の長文は解析されないのでご留意ください。

*3 戻る機能はありません。逐一冒頭のページから解析を行ってください。
